# homebrew-screenpath

Homebrew tap for installing [ScreenPath](https://github.com/Ekwinder/screenpath).

## Install

```bash
brew tap Ekwinder/screenpath
brew install --cask screenpath
```

## Notes

The cask expects a GitHub release asset at:

- `https://github.com/Ekwinder/screenpath/releases/download/v0.2/ScreenPath.app.zip`

Before the first public install works end-to-end, publish a GitHub release for `ScreenPath` and upload `ScreenPath.app.zip`.
